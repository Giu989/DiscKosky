# DiscKosky
A package for analysing the repeated discontinuity structure of Feynman integrals.

Giulio Crisanti, Luke Lippstreu, Andrew J. McLeod and Maria Polackova (2026)

# Early Access Note
Package is in public beta. No documentation is provided.

### Automatic Installation
To install, open a Mathematica window and run the command
```wolfram
ResourceFunction["GitHubInstall"]["giu989","DiscKosky"];
```
The package should now be loadable with
```wolfram
<<DiscKosky`
```
If required, the installation directory can be found with
```wolfram
"Location" /. PacletFind["DiscKosky"][[1]][[1]]
```
